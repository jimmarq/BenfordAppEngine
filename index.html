Benford's Law helps determine if a set of numbers are truly random. Read more about it. Use the form below to analyze your data.

Currently, two types of analysis are offered:

  • Single Digit Analysis. Use this to analyze a list of numbers. For example, it can be used to verify the randomness of a list of invoice amounts.
  • Grouped Single Digit Analysis. This is useful if a list of numbers needs to be analyzed with another variable. For example, a list of invoice amounts and the people who created the invoices could be analyzed.

Single Digit Analysis

Only a single column of numbers is permitted. Leading zeros and any non-digits characters will be ignored. The first digit in a number is most important in determining if it is random. A larger data set will yield more accurate results. A maximum of 10,000 numbers will be analyzed.
